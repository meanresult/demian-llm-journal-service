| 타입       | 의미           | 언제 쓰는가            |
| -------- | ------------ | ----------------- |
| feat     | 새로운 기능 추가    | API 추가, 기능 구현     |
| fix      | 버그 수정        | 오류 수정             |
| docs     | 문서 변경        | README 수정         |
| refactor | 코드 구조 개선     | 기능 변화 없음          |
| test     | 테스트 코드 추가/수정 | 테스트 작성            |
| chore    | 설정, 환경 변경    | Docker, gitignore |
| style    | 코드 포맷 변경     | 들여쓰기 등            |
